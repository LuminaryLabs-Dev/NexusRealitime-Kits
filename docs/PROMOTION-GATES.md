# Promotion Gates

A ProtoKit can become an official NexusRealtime Kit only when it passes these gates.

## Required Gates

- Domain is clear.
- Name follows kit naming grammar.
- Behavior is reusable beyond one game.
- Runtime state is deterministic where state matters.
- Renderer and host behavior are decoupled.
- README exists.
- Install example exists.
- Domain bundle export exists.
- CDN path is documented.
- Test or smoke plan exists.

## Outcomes

```txt
promote
split
merge
rename
keep incubating
archive
```

Promotion is not copying files. Promotion is stabilizing a domain contract.
