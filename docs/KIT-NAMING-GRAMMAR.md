# Kit Naming Grammar

Name by domain meaning, not programming category.

## Good names

```txt
spatial-interaction-kit
completion-ledger-kit
camera-state-kit
render-layer-kit
generic-defense-map-dsk
```

## Avoid vague names

```txt
helpers
utils
common
misc
feature
whole-game-kit
```

## Pattern

```txt
<domain>/<capability>-kit
<domain>/<capability>-dsk
<domain>/<capability>-adapter
<domain>/<theme>-preset
```

Each name should tell an agent what domain owns it, what capability it exposes, and whether it installs, adapts, configures, or bridges.
